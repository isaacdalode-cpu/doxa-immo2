# Doxa Immo 🏠

Plateforme immobilière simple pour la RDC.

## Projet débutant
Ce projet contient :
- HTML
- CSS
- JavaScript

## Objectif
Créer une plateforme immobilière où :
- Les propriétaires publient leurs biens
- Les clients trouvent des logements
- Les agents reçoivent des contacts (leads)
